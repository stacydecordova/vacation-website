CSS Option – Stacy DeCordova

Assignment: Create a website using six CSS features from Chapters 5 and 6

Chosen Option

I chose the CSS option. For this assignment, I created a website about my dream summer vacation to Costa Rica.

What I Did

The site features a homepage with sections on Costa Rica, a scrolling photo gallery, activities list, and a helpful resource link. It demonstrates six different CSS techniques.

CSS Techniques Used

  1. @font-face: A custom font is applied to the entire website for a tropical vibe.
  2. Horizontal Scrolling: The gallery scrolls sideways using flexbox and overflow-x.
  3. Attribute Selector: External links starting with “https” are styled using an attribute selector.
  4. Structural Pseudo-class: The last item in the activity list is styled differently using li:last-child.
  5. Multi-column Layout: The About section uses a two-column layout.
  6. Alpha Transparency: RGB values are used to create semi-transparent backgrounds and shadows.
